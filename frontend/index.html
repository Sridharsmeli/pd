Front-End

Frontend development is the process of creating the part of a website or web application that a user sees and interacts with directly. It is often referred to as the "client-side" of development, in contrast to the "server-side" or backend, which handles data storage and processing.

In this there are

HTML

HTML (HyperText Markup Language) HTML provides the basic structure and content of a web page. It defines the fundamental layout and organization of all the visual elements a user sees.

CSS

CSS (Cascading Style Sheets) CSS controls the presentation and styling of the HTML elements, making a web page visually appealing. It dictates the colors, fonts, spacing, and layout.

javascript

JavaScript is a full-fledged programming language that adds interactivity and dynamic functionality to a web page. It allows developers to manipulate HTML and CSS in response to user actions.

BOOTSTRAP

Bootstrap is a popular, open-source framework that bundles pre-written HTML, CSS, and JavaScript components. It's designed to help developers build responsive, mobile-first websites faster and more efficiently.

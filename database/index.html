DATA-BASE

A database is an organized collection of data, typically stored electronically in a computer system. A back-end system for a web application commonly uses a database for storing, managing, and retrieving the application's data

SQL

Structured Query Language (SQL) SQL is a domain-specific programming language designed for managing data in a relational database management system (RDBMS).

Mongo DB

MongoDB is a NoSQL (non-relational) database program that stores data in flexible, JSON-like documents with dynamic schemas.
